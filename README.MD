# Pera - Bookmark Manager

A minimal, dark-themed bookmark manager that lets you save, search, and visit your favorite websites - all from one clean interface. No accounts, no clutter, no distractions.

## Features

* 🔍 **Search instantly** through all your saved links
* 📥 **Import bookmarks** from a backup file
* 📤 **Export bookmarks** for backup or migration
* 💾 **Offline-ready** - your data stays local
* 📱 **Responsive design** - works on desktop and mobile

## Getting Started

No build tools or dependencies needed - just open and use!

```bash
# Clone the repository
git clone https://github.com/ayushxpundir/Pera.git

# Navigate into the project
cd pera/Resources
```

Then open `index.html` directly in your browser - that's it!

> Note: For the best experience, you can also use the Live Server extension in VS Code.

## Usage

1. **Add a bookmark** - Paste a URL and Pero will auto-fill the title and description.
2. **Search** - Use the search bar to find any bookmark instantly by title, URL, or tag.
3. **Export bookmarks** - Download all your saved bookmarks as a backup file.
4. **Import bookmarks** - Restore bookmarks from a previously exported backup file.

## Tech Stack

* **Frontend:** Vanilla HTML, CSS, JavaScript
* **Storage:** Browser localStorage (your bookmarks stay on your device)
* **Data Management:** Import/Export via JSON files
* **Styling:** Custom CSS with responsive media queries
